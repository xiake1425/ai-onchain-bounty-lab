第一目标

Tether.dev Grants & Bounties

官方公开赏金，付款单位是 USD₮。优先看 WDK 模板、浏览器扩展、eCommerce 支付 demo 这类可交付任务。

  • 金额：1,500-10,000 USD₮
  • 动作：先做申请材料和相关经验页
  • 边界：申请前不授权钱包、不支付费用
优先

AI Agent / Web3 黑客松

用 Codex 做一个小而完整的 demo，提交视频、说明文档和代码仓库。

  • 适合：开发原型、写文档、录演示
  • 收入：奖金、赞助、后续外包线索
  • 动作：每天查 ETHGlobal、DoraHacks、生态官网
稳妥

中文教程 / 项目文档

给海外 Web3 项目做中文教程、入门页、FAQ、部署指南和社区素材。

  • 适合：资料整理、翻译、本地化表达
  • 收入：USDT 赏金、项目方打赏
  • 动作：主动给项目方发样稿
可产品化

链上数据小看板

做简单看板：钱包观察、空投任务进度、项目公告聚合、风险提醒。

  • 适合：前端页面、API 聚合、自动摘要
  • 收入：订阅、赞助、工具授权
  • 动作：先做公开样例，不接用户私钥

Rules

实验边界

  • 页面只展示公开地址，不保存私钥、助记词或交易所密码。
  • 不做自动交易、不带单、不承诺收益。
  • MetaMask 只负责收款展示；任何转账、授权、签名都必须人工确认。
  • 不碰需要预付高额费用的任务。
  • 每个机会先做骗局检查，再决定是否投入时间。
  • 所有收入和失败都记录，方便复盘。
