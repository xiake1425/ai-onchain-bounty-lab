# Candidate Projects for First Outreach

Updated: 2026-06-12

Purpose: find 10 projects where Good luck can offer a small free sample first, then try to convert replies into 20-50 USD / USDT micro tasks.

Rule: do not connect wallets, sign messages, send funds, download unknown software, or ask for private keys. The first offer is only a page/doc/product review.

## Best First 3

### 1. Lyra Web3 Playground

Link: https://github.com/nirholas/lyra-web3-playground

Why it fits:

- Small enough to get noticed.
- Product is beginner-facing, so onboarding feedback is useful.
- README says it is a Web3 IDE with tutorials and templates.

Free sample idea:

- Review the homepage or README from a beginner's point of view.
- Deliver 5-8 suggestions: unclear terms, missing steps, first-run friction, doc structure.

Message to use:

- Template 1: Product Testing Feedback
- Or Template 2: Documentation Cleanup

### 2. Passkeys 4337 Smart Wallet

Link: https://github.com/passkeys-4337/smart-wallet

Live demo: https://passkeys-4337.vercel.app

Why it fits:

- It is about smart wallets and passkeys, which is hard for normal users to understand.
- Public demo exists, so we can review without asking for access.
- Good target for onboarding and risk-note feedback.

Free sample idea:

- Review the live demo and README.
- Make a one-page "first-time user notes" report.

Message to use:

- Template 1: Product Testing Feedback
- Template 3: Web3 Task Summary

### 3. Wander Docs

Link: https://github.com/wanderwallet/wander-docs

Why it fits:

- Docs project, very suitable for cleanup and user guide suggestions.
- Wallet-related but we can stay at documentation level.

Free sample idea:

- Pick one doc page and rewrite the opening section to make it friendlier for beginners.
- Add a short "do not share private keys" user warning if relevant.

Message to use:

- Template 2: Documentation Cleanup

## First Outreach Plan

1. Start with Lyra Web3 Playground, Passkeys 4337 Smart Wallet, and Wander Docs.
2. Send only the free mini-review message first.
3. Do not mention Payoneer in the first message.
4. If they reply positively, offer a paid follow-up at 20-30 USD / USDT.
5. If payment is discussed, use Wise or USDT first while Payoneer is pending.

## Next Free Sample Format

Each sample should be short:

- Project name
- Page reviewed
- What I tried
- 3 things that were clear
- 5 things that could be clearer
- Suggested rewrite for one small section
- Safety note if wallet-related
