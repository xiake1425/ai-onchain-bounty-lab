## Hard Rules

Only consider tasks that pass these rules:

- No deposit / recharge / top up.
- No staking / lending / borrowing / liquidity provision.
- No real trading, perps, prediction markets, or leverage.
- No private key, seed phrase, exchange password, or verification code.
- No unknown software downloads.
- No wallet signature unless the exact action is reviewed first.
- No KYC/passport unless the user explicitly chooses it later.
- Use official websites and official social links only.

If a task requires a wallet, use a separate low-value wallet only. Do not use the user's main wallet.
